# Simple E-commerce Store

A lightweight, functional online store project built with **Vanilla JavaScript**. This project demonstrates core web development concepts like DOM manipulation, state management, and event handling without using any external frameworks or libraries.

## 🚀 Features
*   **Product Gallery:** Dynamic rendering of products from a data object.
*   **Shopping Cart:** Add, remove, and update item quantities.
*   **Total Calculation:** Real-time price updates in the cart.
*   **Search & Filter:** Find products by name or category (using the search bar).
*   **Responsive Design:** Optimized for both desktop and mobile devices.

## 🛠️ Tech Stack
*   **HTML5:** Semantic structure.
*   **CSS3:** Custom styling and Flexbox/Grid layout.
*   **JavaScript (ES6+):** Business logic and UI updates.
